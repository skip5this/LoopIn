Close the loop.

Vibe coding has a blind spot. You can see the bug, but your coding agent can't. So you describe it in words, paste screenshots, copy selectors by hand. LoopIn lets you point at the problem and send it directly — DOM, styles, selector, and your instructions — in one click.

How it works

01

Click

Enable capture mode. Click any element on any page.

02

Instruct

Add context. Tell your agent what to do with it.

03

Ship

Element data flows to Claude Code via MCP. Agent acts on it.

Everything your agent needs.

</>

Setup in 3 minutes

01

Clone the repo and install: npm install && npm run build

02

Add the MCP server to your ~/.claude.json config

03

Load the extension/ folder as an unpacked Chrome extension

04

Click any element. Ask Claude Code: "what did I just capture?"

Bridge the gap.

Your browser and your coding agent, finally connected.
